# 🚗 EV Market Intelligence Dashboard

## Project Overview

This Power BI dashboard provides a comprehensive analysis of India's Electric Vehicle (EV) market. The dashboard focuses on EV registrations, manufacturer performance, vehicle categories, charging infrastructure, and state-wise market trends.

The project transforms raw EV market data into meaningful insights using interactive visualizations and business intelligence techniques.

---

## Objectives

- Analyze EV market growth trends.
- Identify top EV manufacturers.
- Compare vehicle category registrations.
- Evaluate charging infrastructure across states.
- Analyze state-wise EV ecosystem development.
- Generate actionable insights through interactive dashboards.

---

## Dataset Information

The project uses the following datasets:

- EV Maker by Place
- EV Category Data (2001–2024)
- EV Sales by Makers and Categories (2015–2024)
- Operational Public Charging Stations (PCS)
- Vehicle Class Registrations

Source: Kaggle

## Key Insights

- Maharashtra leads in EV ecosystem development.
- Ola Electric is among the leading EV manufacturers.
- Three-Wheeler segment dominates the EV market.
- EV adoption has grown significantly in recent years.
- Charging infrastructure is concentrated in major states.
